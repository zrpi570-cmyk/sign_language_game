# 手语星球 - Sign Language Planet

> **让每一双手都能说话，让每一个声音都被看见。**
>
> 一款基于 DeepSeek AI + Three.js 3D 地球的中国手语(CSL)沉浸式平台。

---

## 项目简介

手语星球是一个面向中国手语(Chinese Sign Language, CSL)学习者的全栈 Web 应用。它将 1098 个手语词汇视频与 DeepSeek AI 智能分类、Three.js 3D 地球可视化、沉浸式阶段学习、闯关答题、摄像头手势识别等多种交互形式融为一体，让手语学习变得有趣、直观、高效。

项目以 Flask 为后端框架，SQLite 为本地数据库，前端采用纯原生 HTML/CSS/JS 配合 Three.js 构建沉浸式 3D 场景，所有词汇的词性与语义分类均由 DeepSeek Chat API 精确标注。

---

## 核心功能

### 首页 - 3D 地球互动
- Three.js 实时渲染的地球纹理，带大气辉光、星群粒子与轨道环
- 92 个城市标记点(含全部中国省会 + 全球主要城市 + 全部非洲国家首都)
- 5 种城市图标形状:圆、星、钻石、爱心、环
- 鼠标拖拽旋转视角、滚轮缩放(10-50 倍)
- 城市名称悬浮标签 + 淡光连接线

### 阶段学习系统 - 12 个主题关卡
按语义分类划分的渐进式学习路径，每个关卡包含"学习 > 答题 > 通关评价(星星评级)"的完整流程:

1. 你好世界 - 人称称呼
2. 数数大师 - 数字表达
3. 色彩身体 - 颜色与身体部位
4. 时间旅行 - 时间表达
5. 生活百物 - 日常物品、食物饮料、服装配饰
6. 动物世界 - 动物与大自然
7. 动起来 - 常用动作手势
8. 心情日记 - 情感心理、状态感受
9. 品质特征 - 描述人的品质
10. 健康生活 - 医疗健康、社交人际
11. 世界各地 - 国家地名、交通、建筑场所
12. 聋人文化 - 手语聋文化、教育

### 闯关游戏
- 从全部词汇中随机出题,支持按分类/词性/难度/数据集筛选
- 四选一选择模式,实时反馈对错
- 累计经验值(XP)与等级,记录连胜与最高连胜
- 答对 +10 XP,答错 +2 XP

### 视频词库浏览
- 分页浏览全部 1098 个手语词汇视频
- 多维度筛选:分类、词性、难度、数据集
- 点击播放视频 + 查看词性与语义分类详情

### 智能复习系统
- 自动收集所有学习与游戏中的错题
- 按错误次数排序,优先复习最薄弱的词汇
- 集中播放错题视频,逐一标记已掌握

### 摄像头手势挑战(实验性)
- 基于 MediaPipe Hands 的实时手部关键点检测
- 5 个难度递增的关卡:指尖数字 > 手势入门 > 手势进阶 > 双手挑战 > 综合大师
- 在浏览器端完成手势识别,不依赖服务器

### 用户系统
- 用户名/密码注册登录
- 访客快速体验模式
- 用户经验值(XP)与等级体系
- 学习进度、游戏记录、错题本持久化

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端框架 | Python Flask |

| 部署运行 | Python 3.x,直接启动 |

## 本地运行

### 1. 进入项目
```bash
cd D:\\sign_language_game
```

### 2. (可选)导入数据集
如果你有 CSL 视频数据集,放入 dataset/CSL_basic_dataset/ 和 dataset/CSL_common_dataset/,然后运行:
```bash
set PYTHONIOENCODING=utf-8
python utils/dataset_importer.py
```
脚本会自动扫描视频文件导入数据库,调用 DeepSeek API 对每个词汇标注词性和语义分类,统计并打印导入结果。

如需重新 AI 分类所有词汇:
```bash
python utils/dataset_importer.py --ai-reclassify
```

### 3. 启动应用
```bash
set PYTHONIOENCODING=utf-8
python launch.py
```
或
```bash
set PYTHONIOENCODING=utf-8
python app.py
```

### 4. 打开浏览器
访问 http://localhost:5000

## 做这个的初衷

"世界上有两种沉默,一种是无话可说,一种是有话说不出来。"

在中国,有超过 2700 万听障人士,手语是他们与这个世界沟通最重要的桥梁。然而,手语学习的门槛依然很高——线下课程资源稀缺,线上系统化的学习平台更是凤毛麟角。

作为一个热爱技术、也热爱人文关怀的开发者,我一直希望用代码做点什么,让手语学习变得更简单、更有趣、更触手可及。

这就是手语星球诞生的原因。

我希望它不只是一个工具,更像是一个"星球"——一个属于所有手语学习者的星球。在这里,你可以在地球上漫游,感受世界的广阔;从零开始,一步步学习手语;在游戏中巩固记忆;对着摄像头比划,检验自己的手势。

每一个词汇视频,每一次答题,每一颗星星,都是我用心打磨的成果。手语不是沉默的语言,它是看得见的声音。

如果你也认同这个愿景,欢迎关注我的微信公众号"程序猪666",我们一起让手语被更多人看见。
